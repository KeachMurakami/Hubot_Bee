# hubot-bee

A hubot script that does the things

See [`src/bee.coffee`](src/bee.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-bee --save`

Then add **hubot-bee** to your `external-scripts.json`:

```json
[
  "hubot-bee"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
